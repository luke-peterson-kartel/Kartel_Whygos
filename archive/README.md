# Archive Directory

This directory contains historical source files and assets that are not actively used in the application but are kept for reference.

## Contents

### markdown-sources/
Original markdown files containing the 2026 WhyGO goals:
- Company WhyGOs
- Department WhyGOs (Sales, Production, Generative, Community, Platform)

These files were imported into JSON format and are now in `kartel-whygo-system/data/`.

### individual-drafts/
Draft individual WhyGO documents submitted by team members:
- Wayan's draft goals (DOCX and MD formats)

### assets/
Brand assets and images:
- Kartel AI logo (black fill with neon outline)

### convert_to_markdown.py
Legacy script used for converting DOCX files to markdown format.

## Usage

These files are archived for historical reference and should not be modified. The active data is in:
- JSON data: `kartel-whygo-system/data/`
- Knowledge files: Root `knowledge/` directory
